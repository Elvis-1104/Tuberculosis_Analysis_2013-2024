# Tuberculosis_Analysis_2013-2024

This project presents an end-to-end data analysis of global Tuberculosis (TB) trends using WHO-based datasets. The objective is to analyze TB incidence, treatment success rates (TSR), demographic variations, and regional disparities to derive actionable public health insights.

## Objectives

1. Analyze global TB incidence trends (2013–2024).
2. Compare treatment outcome metrics (TSR, TFR, TDR, TLR) across top high-burden countries and WHO regions.
3. Examine association between TB incidence and Treatment Success Rate.
4. Evaluate demographic differences in TSR (age and sex categories).
5. Test whether mean TSR differs significantly across WHO regions using ANOVA.

## Key Insights

1️⃣ Global TB Incidence Trends

1. Overall increasing trend from 2013 to 2024.
2. Noticeable dip in 2020 likely due to COVID-19 under-reporting.
3. South-East Asia (SEA) contributes the highest TB burden.
4. Strong regional disparities in disease distribution.

2️⃣ Treatment Outcome Performance

1. Most high-burden countries maintain TSR ≥ 85%.
2. India and Congo fall in 75–84% range.
3. AFR region shows comparatively higher death rates.
4. Loss to follow-up remains a concern in select countries.

3️⃣ Correlation Analysis

1. Very weak correlation (r = 0.1) between TB incidence and TSR.
2. High disease burden does not automatically imply poor treatment success.
3. Reducing incidence alone will not guarantee better treatment outcomes.

4️⃣ Demographic Analysis

1. TSR remains above 80% across all demographic categories.
2. Females show slightly higher TSR than males.
3. Pediatric group (0–14) has marginally lower TSR.
4. Overall demographic variation is modest.

5️⃣ Statistical Testing (ANOVA)

1. Significant regional differences in TSR (p < 0.001).
2. Null hypothesis of equal regional means rejected.
3. Treatment outcomes vary due to infrastructure and implementation differences.

## Tools and Technologies
1. Python - Pandas, Seaborn, Matplotlib, Numpy and Scipy(Chi-square Hyothesis Test)
2. Jupyter Notebook
3. Canva
4. MS Word

## Conclusion

This project demonstrates how statistical analysis combined with interactive visualization can uncover meaningful public health insights. While global TB treatment programs show strong overall effectiveness, regional inequalities and systemic healthcare gaps continue to influence outcomes.
